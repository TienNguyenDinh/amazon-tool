How to use:

  1. Copy an Amazon product URL from your browser
  2. Paste it into the input field above
  3. Click "Scrape Product" to extract data
  4. View the extracted data in the professional table below
Note: This tool respects Amazon's servers by adding delays between requests. Scraping may take 10-30 seconds depending on the page complexity.

Built with Node.js, Express, and Cheerio
