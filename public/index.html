How to use:

  1. Single Product Mode:
    • Select "Single Product URL" mode
    • Copy an Amazon product URL from your browser
    • Paste it into the input field
    • Click "Scrape Product" to extract data
  2. Multiple Products Mode:
    • Select "Multiple Product URLs" mode
    • Enter one Amazon product URL per line in the text area
    • You can enter up to 10 URLs at once
    • Click "Scrape Products" to extract data from all URLs
    • Use Ctrl+Enter as a shortcut to start scraping
  3. View the extracted data in the professional table below
Note: This tool respects Amazon's servers by adding delays between requests. Scraping may take 10-30 seconds per product depending on the page complexity. Multiple products are processed sequentially.

Built with Node.js, Express, and Cheerio
