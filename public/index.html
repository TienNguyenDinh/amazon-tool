How to use:

  1. Paste Amazon URLs:
    • Copy any Amazon URL from your browser
    • Paste into the text area (single URL or multiple URLs, one per line)
    • The tool automatically detects URL types
  2. Supported URL Types:
    • Product URLs: Individual product pages (e.g., /dp/ASIN)
    • Search URLs: Search results pages (e.g., /s?k=keyword)
    • Category URLs: Best sellers, category pages (e.g., /zgbs/)
    • Store URLs: Brand stores and shop pages
  3. Processing:
    • Click "Magic Button" to start extraction
    • Use Ctrl+Enter as a keyboard shortcut
    • The tool processes each URL with appropriate logic
    • Results are displayed in the table below
Note: This tool respects Amazon's servers by adding delays between requests. Processing time varies: 5-15 seconds per product URL, 15-30 seconds per search/category URL depending on complexity. URLs are processed sequentially for best results.

Built with Node.js, Express, and Cheerio
